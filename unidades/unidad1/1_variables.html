5  Variables y Tipos de Datos en Dart

En Dart, todo es un objeto. Como facilitador, quiero que entiendas que la seguridad de tipos (type safety) no es una restricción, sino una herramienta para evitar errores en producción.

5.1 Tipos de Datos Básicos

Dart soporta los siguientes tipos fundamentales:

  • String: Cadenas de texto.
  • int / double: Números enteros y de punto flotante.
  • bool: Valores lógicos (true/false).
  • List / Map: Colecciones de datos.
DART
void main() {
  final String name = 'Abacom Flutter'; // Inmutable después de la asignación
  const double version = 3.41;          // Constante en tiempo de compilación
  int students = 15;
  
  print('Bienvenido a $name versión $version');
}

5.2 Inferencia de Tipos vs Tipado Explícito

Dart puede inferir el tipo usando var, pero en este curso preferimos el tipado explícito o el uso de final para mejorar la legibilidad y mantenibilidad.

¿Por qué final es preferible a var en el desarrollo de aplicaciones móviles? Investiga la diferencia entre final y const en Dart. ¿Cuál de los dos usarías para definir la URL de una API que se obtiene en tiempo de ejecución?
